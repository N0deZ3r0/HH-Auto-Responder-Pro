# ⚡ HH Авто-Отклик Pro v1.2

Автоматический бот для откликов на вакансии HH.ru


## 📥 УСТАНОВКА

1. Скачайте файлы репозитория
2. Откройте chrome://extensions/
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное"
5. Выберите папку с файлами


## 🌟 ОСНОВНЫЕ ФУНКЦИИ

| Функция | Описание |
|---------|----------|
| 🚀 Скорость | От 0.5 сек/отклик, настраиваемая задержка |
| 🧩 Автофильтр | Автоматическое запоминание организаций |
| 🎭 Темы | Тёмная/светлая тема без перезагрузки |
| 📈 Статистика | Отслеживание успешных/неудачных откликов |
| ⚙️ Настройки | Гибкая конфигурация под себя |
| 🚫 Чат | Автоматическое закрытие окна чата |
| 📝 Письмо | Возможность отключить сопроводительное письмо |

| 🗑️ Очистить статистику | Сброс счётчика откликов |
| 🧹 Очистить автофильтр | Удаление всех организаций из автофильтра |

## 📋 ПОРЯДОК РАБОТЫ СКРИПТА

    1. Закрыть чат (если появился)
              ↓
    2. Найти поле для письма → заполнить → отправить
              ↓
    3. Найти "Добавить сопроводительное" → нажать → повторить
              ↓
    4. Найти "Все равно откликнуться" → нажать → повторить
              ↓
    5. Найти "Откликнуться" → нажать → отправить
              ↓
    6. Закрыть модальное окно


## ⚠️ ВАЖНЫЕ ПРАВИЛА

- Используйте настоящее имя в сопроводительном письме
- Начните с теста (кнопка "Тест на 1 вакансию")
- Не используйте задержку 0с — это может вызвать блокировку
- Откликайтесь релевантно — не спамьте
- Соблюдайте правила HH.ru


## 🛠️ РЕШЕНИЕ ПРОБЛЕМ

| Проблема | Решение |
|----------|---------|
| Не работает | Обновите страницу HH.ru и перезапустите расширение |
| Не отправляется отклик | Увеличьте задержку в настройках |
| Панель не видна | Нажмите на кнопку 🚀 в правом верхнем углу |
| Чат мешает работе | Скрипт автоматически закрывает его |
| Поле письма не заполняется | Убедитесь что галочка "Не отправлять" выключена |
